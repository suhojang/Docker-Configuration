# Docker 서버환경 구성

### 프로젝트 계획 이유
> Docker로 Web Server환경을 손쉽게 하기 위해 DokcerFile/docker-compose.yml 설정 파일 구성

### 사용방법
```
1. 초기 설정
  1) docker-compose.yml이 있는 경로로 이동
    - ex) cd /app/docker/init
  2) docker-compose 실행
    - docker-compose up -d
```
